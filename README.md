\\# Coagulation Analysis Project





This project analyzes coagulation laboratory data and builds models for treatment optimization.





\\## How to Use







1\\. Put your data file in `dir/data.xlsx`



2\\. Install requirements: `pip install -r requirements.txt`



3\\. Open notebook: `jupyter notebook notebooks/main\\\_analysis.ipynb`



4\\. Run the cells in order







\\## Project Structure



\\- `config/`: Settings and configurations



\\- `data/`: Data processing code



\\- `models/`: Machine learning models



\\- `visualization/`: Plotting functions



\\- `notebooks/`: Jupyter notebooks for analysis







\\## Requirements



See `requirements.txt` for Python packages needed.
